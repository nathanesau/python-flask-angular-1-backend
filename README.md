## Heroku

Open repository then

```
git push heroku master
```

to update the API running [https://enigmatic-oasis-17388.herokuapp.com/](here)

## Server logs

To see server log use

```
export HEROKU_APP=enigmatic-oasis-17388
heroku logs -n 100
```
